## Featured projects

> 🚧 Projects currently under active development — check back soon.

### 🚧 [pix-payment-simulator](https://github.com/matheeuspc/pix-payment-simulator) `[in progress]`
> Production-grade simulation of the Brazilian Pix payment flow

Full implementation of QR Code generation (static & dynamic), webhook confirmation, idempotency keys, and rate limiting — modeled after Banco Central's DICT and SPI specifications. Built with **Java 21, Spring Boot 3, PostgreSQL, Testcontainers, Docker Compose**. OpenAPI documented.

### 🚧 [distributed-rate-limiter](https://github.com/matheeuspc/distributed-rate-limiter) `[in progress]`
> Token Bucket and Sliding Window algorithms over Redis

A distributed rate limiter library built on top of Spring Boot + Redis. Supports per-user and per-IP strategies, configurable via annotations. Includes benchmarks and load tests. Designed for high-concurrency fintech APIs.

### 🚧 [event-driven-ledger](https://github.com/matheeuspc/event-driven-ledger) `[in progress]`
> Financial ledger using Event Sourcing + CQRS

Append-only transaction ledger with full audit trail, event replay capability, and separated read/write models. Uses **Kafka, PostgreSQL, Spring Boot 3**. Domain: multi-currency account balances.
